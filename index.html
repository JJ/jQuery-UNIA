Qué es jQuery

  • Librería de JavaScript
  • Simplifica la creación de programas e interfaces ricos de usuario.
  • Creado en el año 2006.
  • Usado en la mitad de los sitios más populares (más que Flash).
  • La mejor librería para manipulación del DOM, Ajax y efectos visuales.

Software libre, por supuesto

  • Usa una licencia MIT
  • Descargable desde jQuery.org.
  • Y usable desde ahí o desde Google hosted libraries
  • O, por supuesto, desde tu propia copia en tu propio sitio web (porque es libre).
  • Abstrae las diferentes implementaciones de los navegadores.

Base de un ecosistema de módulos

  • jQuery UI, para creación de interfaces de usuario.
  • jQuery mobile, para móviles y basada en HTML5
  • Y plugins para jQuery
  • Aparte de jmpress.js, con el que estoy haciendo esta presentación.

El dólar se paró aquí

ready.html

<html> <head> <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-15"> <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script> <title>Probando ready de jQuery</title> </head> <body> <h1>Esta es una página que no tiene gran cosa</h1> <p>Pero podría tenerla.</p> <script type='text/javascript'> $(document).ready(function() { alert('Ahora estamos listos'); }); </script>

Claves del programa anterior

  • Inclusión del jQuery alojado por Google.
  • $ es un identificador válido de JavaScript y una abreviatura de jQuery
  • $(document) == jquery(document) llama a una función sobre el objeto document.
  • ... que devuelve otro objeto sobre el que creamos un evento cuando el documento está ready.

Creando una agenda

jQuery crea eventos: cuando suceda esto (ready), haz esto otro (alert): estilo asíncrono

Tan simple, tan simple...

  • jQuery usa extensivamente las funciones anónimas o closures de JavaScript
  • Hemos creado un evento que se lanza cuando el documento está cargado
  • Se crea una función anónima a la que se llama asíncronamente
