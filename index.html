Tutorial de jQuery y jQuery Mobile

Por JJ Merelo

Curso Desarrollo de aplicaciones multiplataforma para dispositivos móviles, UNIA, La Rábida, Huelva, julio 2013

Qué es jQuery

  • Librería de JavaScript
  • Simplifica la creación de programas e interfaces ricos de usuario.
  • Creado en el año 2006.
  • Usado en la mitad de los sitios más populares (más que Flash).
  • La mejor librería para manipulación del DOM, Ajax y efectos visuales.

Software libre, por supuesto

  • Usa una licencia MIT
  • La licencia MIT es permisiva y permite incluirlo dentro de software privativo
  • Eso no significa que debas hacerlo.

Donde conseguirlo

  • Descargable desde jQuery.org.
  • Y usable desde ahí o desde Google hosted libraries o cualquier otra Content Delivery Network (CDN).
  • Uso minificado: descarga rápida.
  • O, por supuesto, desde tu propia copia en tu propio sitio web (porque es libre).
  • Abstrae las diferentes implementaciones de los navegadores.

    • Base de un ecosistema de módulos

    • jQuery UI, para creación de interfaces de usuario.
    • jQuery mobile, para móviles y basada en HTML5
    • Y plugins para jQuery
    • Aparte de jmpress.js, con el que estoy haciendo esta presentación.

    Claves del programa anterior

    • Inclusión del jQuery alojado por Google.
    • $ es un identificador válido de JavaScript y una abreviatura de jQuery
    • $(document) == jquery(document) llama a una función sobre el objeto document.
    • ... que devuelve otro objeto sobre el que creamos un evento cuando el documento está ready.

    Creando una agenda

    jQuery crea eventos: cuando suceda esto (ready), haz esto otro (alert): estilo asíncrono

    Tan simple, tan simple...

    • jQuery usa extensivamente las funciones anónimas o closures de JavaScript
    • Hemos creado un evento que se lanza cuando el documento está cargado
    • Se crea una función anónima a la que se llama asíncronamente

    Recuerda, jQuery es siempre asíncrono

    Ejercita lo aprendido (Bloque 1)

    Crear una página que al cargarse realice alguna acción, como por ejemplo extraer usando JS todos los títulos H2 de secciones o la primera palabra de cada párrafo. Alternativamente, imprimir el número de elementos de un tipo dado (H2), por ejemplo.

    Seleccionando partes del documento

    • Se usan selectores CSS3 entre comillas: $("h2"), por ejemplo.
    • Un selector devuelve un set jQuery. un objeto sobre el cual se puede aplicar otra función: 
    $("h2").each( function() { 
    alert(this.text);
}

    Ejemplo: Cambiar color a rojo

    Unas palabrejas cualesquiera

    $("#a_rojo form").submit(function(){
    $("#a_rojo").css( "color", "red" );
    return false;
});

    Las llamadas a selectores devuelven el mismo conjunto jQuery

    Unas palabrejas cualesquiera

    $("#a_rojo_2 form").submit(function(){
    $("#a_rojo_2")
	.css("color", "red" )
	.css("background-color", "blue" );
    return false;
});

    Aunque en este caso se puede pasar JSON

    Otras palabrejas cualesquiera

    $("#a_rojo_3 form").submit(function(){
    $("#a_rojo_3").css( { color: "brown",
			  backgroundColor: "yellow" } );
    return false;
});

    Un CSS un poco especial: notación JS

    • Se usa backgroundColor en vez de background-color.
    • Claves sin comillas, valores con comillas.
    • jQuery admite CSS 1 a 3

    Podemos usar también notación CSS (entre comillas)

    Otras palabrejas cualesquiera

    $("#a_rojo_4 form").submit(function(){
    $("#a_rojo_4").css( { "font-weight": "bold",
			  "font-size" : "2em" } );
    return false;
});

    Metiendo y sacando cosas

    Aparte de html, que mete código a cascoporro, tenemos

    • after: añade código después del elemento (siguiente en el DOM) y append después del contenido del elemento.
    • prepend lo añade antes del contenido del elemento.
    • remove elimina un elemento, mientras que clone lo copia.

    Más texto por todos lados

    (^_^) 

    $("#texto form").submit(function(){
    $("#texto p").prepend( "Mira: ")
      .append(" - ¡Qué feliz!")
      .after('<p>Este no: |-)</p>');
    return false;
});

    También nos lo podemos cargar

    (^_^) 

    $("#clona").submit(function(){
    $("#texto_2 p").after( $("#texto_2 p").clone());
    return false;
});
$("#borra").submit(function(){
    $("#texto_2 p").remove(); return false;
});

    ¿Cómo funciona clone?

    • Crea un nuevo elemento, pero desligado del resto del DOM.
    • Por eso tenemos que usar after para indicarle dónde colocarlo.

    Podemos ser más selectivos

    (^_^) 

    $("#clona_2").submit(function(){
$("#texto_3 p").after( $("#texto_2 p").clone()
  .addClass("nuevo").css("color","green")); return false;
  });
$("#borra_2").submit(function(){
  $("#texto_3 p.nuevo").remove(); return false;
});

    Clonando con clase

    • addClass añade un atributo de clase al elemento que se crea
    • Ese atributo de clase se usa para seleccionar el elemento (o elementos) que queremos borrar.
    • ¿Qué pasaría si se pulsa varias veces el botón Clona? ¿Por qué?

    Siempre hay más de una forma de hacerlo

    (^_^) 

    $("#clona_3").submit(function(){
      $("#texto_4 p")
        .after( $("#texto_4 p").clone()
        .attr("class","nuevo").css( { "color" : "DarkBlue",
                                      "font-weight": "Bold" }));
      return false;
}); // Resto igual

    Creando un calendario de eventos

    • Ya hemos visto uno de los eventos: ready
    • on es la función más general para asignación de eventos.
    • La función callback recibe un objeto con el evento; la variable this contiene el elemento sobre el que ha sucedido el evento.

    Haciendo que pasen cosas

    (^_^) 

    $("#evento").on("click", function(){
		    $("#evento p").text("😇")
      .css({ "background-color": "lightblue", 
      color: "yellow",
      fontSize: "3em" } );
});

    Delegado en funciones

    (^_^) Eres un ángel 

    $("#evento_2").on("mouseover", "span", function( event ){
		      $("#evento_2 p").text("😇")
			  .css({ "background-color": "yellow", 
				 color: "lightblue",
				 fontSize: "3em" } );
});

    Navegando por un mar de objetos

    • Partiendo de los selectores anteriores, jQuery permite trabajar con los nodos del DOM adyacentes.
    • parent() accede al nodo padre, next() y prev() al siguiente y anterior, children() a todos los hijos y .siblings() a los hermanos .
